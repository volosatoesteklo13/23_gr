Для выполнения задания у вас должен быть установлен для Windows - GitBash.
Создан аккаунт в GitHub
Создан репозиторий в GitHub
Созданный репозиторий склонирован на локальный компьютер в отдельную папку.

Все шаги сценария выполняйте в терминале GitBush, Terminal, в папке под гитом.

Как отправить ДЗ на проверку.
 1. Создайте текстоовый файл как в первом ДЗ по Terminal.
 2. Сценарий перенесите в этот файл.
 3. На против каждого действия - напишите команду в GitBash

Файл со сценарием и ссылку на свой гит хаб отправляйте менторам на проверку.

 4. Создать файл “new.json” touch new.json
 5. Добавить файл под гит.  git add .
 6. Закоммитить файл.       git commit -m "home2"
 7. Отправить файл на внешний GitHub репозиторий. git push
 8. Создать файлы new2.json, new2.xml, new2.txt. touch new2.json new2.xml new2.txt
 9. Добавить файлы под гит. git add .
 10. Закоммитить файлы.   git commit -m "home2.1"
 11. Отправить файлы на внешний GitHub репозиторий. git push
 12. Отредактировать содержание файла “new2.txt” - написать информацию о себе (ФИО, возраст, количество домашних животных, будущая желаемая зарплата).vim new2.txt пишу"Коневич Конь Бенгалович 30 коньских лет 0 животных триллион минимум" :wq
 13. Отредактировать содержание файла “new2.json” - написать информацию о себе (ФИО, возраст, количество домашних животных, будущая желаемая зарплата). Всё написать в формате JSON.vim new2.json пишу:
{ 
  "Ф_И_О": "Коневич_Конь_Бенгалович", 
  "возраст": 30, 
  "Количество_животных": "Я_же_конь_какие_животные", 
  "Желаймая_зарплата": 100000000000000000000000000000000,

 14. Добавить и закоммитить “new2.txt” одной строчкой.  git commit -am "txt_and_json_text"
 15. Добавить и закоммитить “new2.json” одной строчкой. git commit -am "txt_and_json_text" 
 16. Отправить изменения на внешний репозиторий. git push
 17. Отредактировать содержание файла “new2.xml” - написать информацию о себе (ФИО, возраст, количество домашних животных, будущая желаемая зарплата). Всё написать в формате XML.
<?xml version="1.0"?>

<возраст>30</возраст>
<Количество_животных>Я_же_конь_какие_животные</Количество_животных>
<Желаймая_зарплата>100000000000000000000000000000000</Желаймая_зарплата>
 18. Добавить и закоммитить “new2.xml” одной строчкой. git commit -am "xml_text"
 19. Отправить изменения на внешний репозиторий.git push
 20. Отправить предыдущее домашнее задание по GitBush Terminal на внешний репозиторий.git commit -am "home" git push
 21. В веб интерфейсе GitHub создать файл new3.txt. жмем добавить файл, пишем название, комментарий по желание, жмем создать

 24. Синхронизировать внешний и локальный репозитории (слить изменения с внешнего репозитория) git pull
